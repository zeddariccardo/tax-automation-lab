# Pubblicazione v3

Questo ZIP contiene esclusivamente il sito da pubblicare su GitHub Pages.
Non contiene PDF, ZIP o HTML distribuiti come download dal dominio.

1. Caricare tutti i file nella root del repository, sostituendo quelli esistenti.
2. Pubblicare separatamente gli asset dello ZIP release su GitHub Releases come v1.0.1.
3. Verificare `/guide/`, `/privacy/`, `/security/`, `/terms/`, `/licenses/` e `/.well-known/security.txt`.
4. Solo dopo i test configurare gli header Cloudflare.
