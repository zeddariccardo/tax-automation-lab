Security policy

Sicurezza

Come funzionano i tool, come segnalare problemi e quali limiti considerare.

Elaborazione locale

I tool sono progettati per elaborare i file nel browser. Non richiedono account, autenticazione o installazione di software e non inviano i documenti caricati a un database del sito.

Segnalazioni

Per segnalare una vulnerabilità o un comportamento inatteso, scrivi a security@taxautomationlab.com. Includi descrizione, passaggi per riprodurre il problema e impatto osservato. Non inviare dati personali o documenti riservati.

Limiti

Nessun sistema può essere descritto come privo di rischio. I tool devono essere utilizzati con browser aggiornati e gli output devono essere sottoposti a verifica professionale.
